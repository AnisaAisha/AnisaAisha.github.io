Mesh Murammat

  • Category: WebGL
  • Project URL: Mesh Murammat

This project is a web application for visualization, editing, hole detection and hole filling of 3D meshes using NodeJS front-end and C++ backend (using CGAL library).
Please note that this project is currently undergoing several enhancements. Hence, some features may not be completely functional.
